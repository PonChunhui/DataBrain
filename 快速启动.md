# 快速启动指南

## 默认管理员账号

系统启动时会自动创建默认管理员账号：

- **用户名**: `admin`
- **密码**: `admin123`
- **邮箱**: `admin@devops.com`

## 启动步骤

### 1. 启动数据库
```bash
# 使用Docker启动MySQL
docker run -d --name devops-mysql \
  -e MYSQL_ROOT_PASSWORD=password \
  -e MYSQL_DATABASE=devopsdb \
  -p 3306:3306 mysql:8.0
```

### 2. 启动后端
```bash
cd backend
go run main.go
```

启动时会看到以下日志：
```
默认管理员账号创建成功 username=admin password=admin123
默认角色创建成功 role=管理员
默认权限初始化完成
管理员角色分配成功
服务器启动 port=:8080
```

### 3. 启动前端
```bash
cd frontend
npm run dev
```

### 4. 登录系统
访问 http://localhost:5173
- 输入用户名：`admin`
- 输入密码：`admin123`
- 点击登录

## 初始化数据

系统启动时自动创建：
- 1个管理员账号（admin/admin123）
- 1个管理员角色
- 10个基础权限（user:create、user:read等）
- 自动为admin分配管理员角色

## 手动创建账号

如果需要手动创建账号，可以使用注册功能：
- 访问登录页面
- 输入用户名和密码
- 点击"注册新账号"按钮

注册成功后即可使用新账号登录。
