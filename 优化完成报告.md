## WebSocket实时日志功能详情

### 实现内容

**后端**：
- `api/v1/pod_log_ws.go` - WebSocket handler
- gorilla/websocket库
- K8s client-go流式日志API
- 支持follow实时跟踪模式

**前端**：
- `components/PodLogTerminal.vue`
- @xterm/xterm终端模拟器
- @xterm/addon-fit自适应大小
- @xterm/addon-web-links链接支持

### 特性

- **实时输出**：WebSocket推送，follow=true
- **终端主题**：深色背景，VSCode风格配色
- **容器选择**：多容器Pod支持切换
- **控制按钮**：清屏、重连、停止
- **自动重连**：连接断开后最多3次重连
- **大scrollback**：支持10000行历史日志
- **状态显示**：已连接/未连接状态，行数统计

### 使用方式

1. Pod管理页面 → 点击"日志"按钮
2. 选择容器（多容器Pod）
3. 设置显示行数（默认100）
4. 实时查看日志输出
5. 可清屏、重连或停止流

### API接口

```
WebSocket连接：
ws://host/api/k8s/pod/log/stream?cluster_id=X&namespace=Y&pod=Z&container=C&tail_lines=100&follow=true

停止流：
POST /api/k8s/pod/log/stop?session_id=xxx
```
